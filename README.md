# 💰 FinTech Expense Tracker with Data Visualization

A modern, visually appealing **expense tracking web application** built using **React + Tailwind CSS**. This app helps users manage their finances, track transactions, set budgets, and visualize spending patterns through interactive charts.

---

## 🚀 Features

### 🔐 Authentication (Login & Signup)
- Animated authentication UI with glassmorphism design
- Floating blurred background effects
- Mock authentication using localStorage
- Protected routes using AuthContext

---

### 📊 Dashboard
- Central financial overview
- Animated cards using Framer Motion
- Summary cards with hover effects and gradients
- Recent Transactions preview section

---

### 💳 Transactions
- View all income & expenses
- Card-based UI instead of traditional tables
- Color-coded transactions:
  - 🟢 Income (Emerald)
  - 🔴 Expense (Rose)
- Category badges for better readability

---

### 🎯 Budget Management
- Set monthly spending limits
- Dynamic progress bar:
  - 🟢 Green → Safe
  - 🟡 Amber → Warning
  - 🔴 Red → Budget exceeded
- Real-time budget tracking

---

### 📈 Analytics
- Data visualization using Recharts
- Pie Chart → Expense breakdown by category
- Bar Chart → Income vs Expense (last 6 months)
- Custom tooltips with glassmorphism styling
- Consistent Tailwind color theme

---

## 🛠️ Tech Stack

- **Frontend:** React.js
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Charts:** Recharts
- **State Management:** Context API
- **Storage:** localStorage (mock auth)
